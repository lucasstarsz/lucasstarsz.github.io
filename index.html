About FastJ

FastJ is a 2D Java Game Engine. While not complete, it contains many of the features needed to create simple 2D games.

Using FastJ

In order to use FastJ, you'll need the following:
  • A Java 11 JDK, or newer
  • An IDE or code editor that supports Java
    • IntelliJ IDEA
    • Eclipse
    • Apache Netbeans
  • The latest release of the FastJ Game Engine

Games Made with FastJ

Pong

Pong was the first game to be made using the FastJ Game Engine, created by the same person who wrote the engine.

It is open source, and made using version 0.3.0a of the FastJ Game Engine.
