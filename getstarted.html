Getting Started

Checklist

Double-check to make sure you've got everything you need to develop games with FastJ before moving on:
  • The Java 11 JDK, or newer.
  • An IDE or code editor that supports Java.
    • Popular IDEs:
      • IntelliJ IDEA
      • Eclipse
      • Apache Netbeans
    • Popular code editors:
      • Microsoft Visual Studio Code
      • Atom
      • Brackets
You'll also need to know how to code using Java. You don't need a ton of Java knowledge, just a good grasp of the basics.

Don't know any Java? Check out this course.
