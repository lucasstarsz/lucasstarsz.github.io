Getting Started

Checklist

Double-check to make sure you've got everything you need to develop games with FastJ before moving on:
  • The Java 11 JDK, or newer.
  • An IDE or code editor that supports Java.
    • Popular IDEs:
      • IntelliJ IDEA
      • Eclipse
      • Apache Netbeans
    • Popular code editors:
      • Microsoft Visual Studio Code
      • Atom
      • Brackets

Java

You'll also need to know how to code using Java. FastJ is programmed using it, and it expects to interface with Java code. Do note, however, that you won't need an extensive amount of experience using Java to use this engine--just the basics will do.
Don't know any Java? Check out this course.
