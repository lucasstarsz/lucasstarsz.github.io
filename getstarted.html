Getting Started

FastJ Checklist

Make sure you've got all the tools needed to develop games with FastJ before moving on:
  • A Java 11 JDK, or newer
  • A way to write and compile your Java code
    • IntelliJ IDEA
    • Eclipse
    • Apache Netbeans
  • The latest release of the FastJ Game Engine
